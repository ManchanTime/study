# 조회용 샘플 데이터 입력

+ InitDb
  데이터베이스에 필요한 값들을 사전에 넣기 위한 class
  UserA가 JPA1 BOOk, JPA2 BOOK을 각각 1권 2권 구입했다고 가정. 책 값은 각각 10000, 20000
  UserB가 SPRING1 BOOK, SPRING2 BOOK을 각각 3권 4권 구입했다고 가정. 책 값은 각각 20000, 40000
